# SBC Brand Assets

This repository contains the official brand assets and guidelines for SBC.

## Contents

- **Brand Book:** Open `index.html` to view the full visual guide and download assets.
- **Assets:**
  - `assets/icons/`: App icons for iOS, Android, and Web.
  - `assets/social/`: Social media images (Open Graph, Twitter, etc.).
  - `assets/favicons/`: Browser favicons.
  - `assets/LOGO.svg`: Master vector logo.

## Usage

For the best experience, open `index.html` in your web browser.
